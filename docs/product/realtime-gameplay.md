# Realtime Gameplay — DRIFT PURSUIT GRID

## Purpose

The realtime gameplay mode turns DRIFT PURSUIT GRID from a deterministic simulation into an interactive pursuit experience.

It allows the player to:

- control the vehicle live
- react to traffic and incidents
- maintain pursuit pressure
- experience failure and recovery
- feel the system, not just observe it
